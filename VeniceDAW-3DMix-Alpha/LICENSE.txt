DISCLAIMER FOR REVERSE ENGINEERING:

This software is a clean-room reverse engineering implementation of the BeOS
3D Mixer file format (.3dmix). No original source code from Be Inc. was used
or referenced in the development of this software.

All functionality was reconstructed through:
- Binary file format analysis
- Public documentation fragments from Internet Archive
- Haiku BeAPI documentation (open source)
- Independent testing and validation

The .3dmix file format is believed to be part of BeOS R5 (1995-2001), which
is now effectively abandonware. Be Inc. was acquired by Palm Inc. in 2001,
and BeOS was discontinued.

This implementation is provided for educational and archival purposes,
to preserve compatibility with historical BeOS audio projects.
